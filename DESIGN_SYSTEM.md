# Al Abbasia Welfare Trust Website Design System

## Brand Direction

The website presents Al Abbasia Welfare Trust as a credible, local, human-centered welfare organization serving Rawalpindi and nearby communities. The visual language is intentionally restrained: clean white surfaces, measured green accents, quiet borders, real field imagery, and generous whitespace. The design avoids decorative trends so that trust, clarity, and donation intent remain central.

## Color System

- Primary Green: `#0C8A43` for primary actions, section labels, icons, and active navigation.
- Dark Green: `#066536` for hover states, strong labels, and footer accents.
- Royal Blue: `#3C3A99` for secondary brand moments and trust badges.
- White: `#FFFFFF` for cards and page surfaces.
- Background: `#FAFBFC` for the global page background.
- Section Background: `#F4F7F6` for soft content bands.
- Heading: `#1F2937` for high-contrast readable titles.
- Body Text: `#4B5563` for paragraphs and supporting text.
- Border: `#E5E7EB` for cards, dividers, inputs, and navigation outlines.

## Typography

- English headings use Poppins SemiBold.
- English body text uses Inter.
- Urdu pages use Noto Sans Arabic with right-to-left layout for a clean, modern Urdu interface.
- Heading hierarchy remains calm: strong but not oversized, with consistent line-height and spacing.

## Spacing Rules

- Page sections use `88px` vertical spacing on desktop, decreasing to `64px` tablet and `48px` mobile.
- Content containers max out at `1180px`.
- Cards use `24px` internal padding on desktop and `20px` on mobile.
- Grid gaps use `24px` and `32px` depending on density.

## Components

- Buttons are purposeful and compact: green primary, white secondary with green border.
- Cards use white backgrounds, `1px` borders, subtle shadows, and `10px` radius.
- Images use natural crops, soft corners, and descriptive alt text.
- Forms use accessible labels, visible focus states, and generous tap targets.
- Navigation is sticky with a small shadow only after scrolling.

## Layout Rules

- Each page starts with a clear page hero or section header.
- Home page prioritizes: mission, trust indicators, real impact, projects, CEO message, news, gallery, testimonials, donation.
- Urdu pages mirror alignment and flow through `dir="rtl"` and native content, not machine translation.
- Animations are subtle: fade-up reveal, counter increments, image hover lift, navbar transition.

## Accessibility And Performance

- Semantic landmarks are used throughout.
- Keyboard focus is visible on links, buttons, and form controls.
- Decorative motion respects `prefers-reduced-motion`.
- Images use lazy loading except key hero media.
- No Bootstrap, Tailwind, jQuery, React, or Vue are used.
